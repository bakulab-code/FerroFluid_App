# FerroFluid App Support Site

Official GitHub Pages site for **FerroFluid: Magnetic ASMR**.

- Support: https://bakulab-code.github.io/FerroFluid_App/support/
- Privacy Policy: https://bakulab-code.github.io/FerroFluid_App/privacy/
- Japanese support: https://bakulab-code.github.io/FerroFluid_App/ja/support/
- Japanese Privacy Policy: https://bakulab-code.github.io/FerroFluid_App/ja/privacy/

This repository contains only the public support website. The FerroFluid application source is maintained separately.

## Local preview

Serve the repository root with any static HTTP server, then open `index.html`.

## Deployment

Pushes to `main` deploy through the GitHub Pages workflow in `.github/workflows/pages.yml`.
